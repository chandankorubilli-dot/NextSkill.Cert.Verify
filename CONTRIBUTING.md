# Contributing to NextSkill Certificate Verification

Thank you for your interest in contributing to this project!

## How to Contribute

### Reporting Issues
- Check existing issues first to avoid duplicates
- Provide a clear description of the problem
- Include steps to reproduce (if applicable)
- Mention your browser/environment

### Code Contributions
- Fork the repository
- Create a feature branch: `git checkout -b feature/your-feature-name`
- Make your changes with clear, descriptive commits
- Test your changes in multiple browsers
- Submit a pull request with a clear description

### Types of Contributions Welcome
- Bug fixes (frontend UI/logic issues)
- UI/UX improvements
- Documentation improvements
- Accessibility enhancements
- Cross-browser compatibility fixes

### Code Style
- Keep code simple and readable
- Use meaningful variable names
- Comment non-obvious logic
- Follow existing formatting style
- No external dependencies (keep it lightweight)

### Before Submitting
- Ensure no console errors
- Test in Chrome, Firefox, Safari, Edge
- Test on mobile devices
- Verify the verification flow works end-to-end

### What Cannot Be Accepted
- Changes to backend (Apps Script) logic
- Commits containing secrets, API keys, or private data
- Changes that break existing functionality

## Questions?
Open an issue with the label `question` for discussions.

Thank you for contributing!
