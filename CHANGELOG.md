# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-13

### Added
- Initial public release of Certificate Verification module
- Clean, responsive web interface for certificate verification
- JSONP integration with Google Apps Script backend
- Mobile-friendly design using Cormorant Garamond and DM Sans typography
- Real-time certificate validation against Google Sheets database
- Support for displaying: Certificate ID, Student Name, Issue Date, Program, Batch, Issuer, Status
- Loading states and error handling
- Comprehensive documentation and security guidelines

### Features
- Public certificate verification page
- Professional UI with gold and green accent colors
- CORS-safe JSONP implementation
- Responsive design (desktop and mobile)
- Clean GitHub Pages deployment ready

### Security
- Backend security checklist in documentation
- Rate limiting guidelines
- Data output control specifications
- Access control best practices

## Versioning

We follow [Semantic Versioning](https://semver.org/).

- **MAJOR**: Breaking changes to functionality or API
- **MINOR**: New features (backward compatible)
- **PATCH**: Bug fixes and improvements
